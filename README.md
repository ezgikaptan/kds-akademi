# 🎭 KDS Akademi Website

A modern, responsive and interactive website developed for **KDS Akademi**, showcasing courses, instructors, events and academy information with a clean, engaging user experience.

## 🌐 Live Demo

🔗 https://kds-akademi.vercel.app/

---

## 📖 About the Project

KDS Akademi is a modern corporate website designed for an art academy. The project focuses on providing an attractive user experience while presenting academy programs, instructors, galleries, events and contact information.

The website was redesigned with a modern UI/UX approach, responsive layouts, smooth animations and optimized performance.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern UI/UX
- 🎭 Academy & Course Showcase
- 👨‍🏫 Instructor Pages
- 🖼️ Gallery Section
- 📝 Registration Page
- 📞 Contact Page
- ⚡ Smooth Animations
- 🚀 Optimized Performance
- 🌐 Cross Browser Compatibility
